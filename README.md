## Installation and Usage

### Requirements:

* Node.js installed
* Firebase's account
### Steps:
1. Clone repo on your local machine:
```
$ git clone https://github.com/mkavo92/chat-room
```
2. Install dependencies:
```
$ cd chat-room
$ npm install
```
3. Execute the app:<br/>
```
$ npm start
```

4. App now running on localhost:3000
